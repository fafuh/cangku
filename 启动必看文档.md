## 环境要求

- Node.js (推荐 v14.x 或更高版本)
- npm (推荐 v6.x 或更高版本)
- 浏览器 (推荐 Chrome, Firefox 等现代浏览器)

## 安装步骤

### 1.安装常见错误指南

https://www.yuque.com/ruanjiankaifa-bto3j/hia9i4/bhxif35zyfdx4mgh?singleDoc# 《项目运行常见错误》

### 2. 安装依赖

在项目根目录下打开命令行工具，执行以下命令安装项目依赖：

```bash
npm install
```

### 3.开发环境启动

```bash
npm run serve
```

